## ðŸš€ How to Run

1. **Install Sass** (if not already installed):

   ```bash
   npm install -g sass
````

2. **Compile SCSS** to CSS:

   From the root directory:

   ```bash
   sass src/scss/main.scss dist/main.css
   ```

   Or with npm:

   ```bash
   npm run build
   ```

3. **Open `src/index.html`** in your browser.
